# Traveling Salesperson Problem (TSP) Solver

## Purpose
The Traveling Salesperson Problem (TSP) is a classic optimization problem where the goal is to find the shortest possible route that visits each city exactly once and returns to the origin city. This document provides details on how to use the TSP solver implemented in our system.
