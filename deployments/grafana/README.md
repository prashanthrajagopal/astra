# Grafana Dashboards

This folder contains versioned Grafana dashboards for Astra Phase 5.

Dashboards:

- `dashboards/cluster-overview.json`
- `dashboards/agent-health.json`
- `dashboards/cost.json`

## Import

1. Open Grafana -> Dashboards -> Import.
2. Upload each JSON file in `dashboards/`.
3. Select your Prometheus datasource.

Optional provisioning file is in `provisioning/dashboards.yml`.
